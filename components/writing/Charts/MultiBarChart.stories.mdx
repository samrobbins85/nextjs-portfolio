# MultiBarChart

`padding` determines the padding around the chart that sits within the axes, so the higher the value the further the column will be away from the y axis etc.

`legendPos` determines the x position of the legend, the larger this is, the further right

`offset` is the distance between the two (or more) bars in each section

`axisLabel` is the label to be shown on the y axis
