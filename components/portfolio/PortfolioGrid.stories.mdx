# PortfolioGrid

Shows a list of portfolio items. The special category "All" will show all items, otherwise, it will show the items that match with that category.

Portfolios have the interface:

```
interface Portfolio {
  categories: string[];
  title?: string;
  slug: string;
  description?: string;
}
```
