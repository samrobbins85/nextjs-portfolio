# Timeline

This is used on the about page to show items in the timeline

Category can be

- `Event`
- `Education`
- `Hackathon`
- `Award`
- `Speaking`
- `Writing`
- `Volunteering`

If the category provided isn't one of these, then the icon of angle brackets will be used
